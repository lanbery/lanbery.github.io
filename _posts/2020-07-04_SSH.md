## Add common user ssh Commands 

> Add user 

```bash 
adduser xxx  # 
usermod -d /home/xxx -G wheel xxx # 设置用户的工作目录
passwd xxx   # 设置用户的密码
```

#AllowUsers<用户名1> <用户名2> <用户名3> ...##指定允许通过远程访问的用户，多个用户以空格隔开
