# Vue Router

> Vue 官方推荐的路由

## 坑

> 传参

  - this.$router.push 和this.$route.params  头一次用容易忽略
  - param 传参  用name:"" 不要用 path

```js
  this.$router.push({
    // path:`/domain/regist`,
    name:"domain.regist",
    params:{
      id:this.searchText
    }
  })
```
